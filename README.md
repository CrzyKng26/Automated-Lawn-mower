# Automated-Lawn-mower
This project aims to develop an advanced automatic lawn mower featuring two modes of operation: fully automated and IoT control via mobile phones. 
The mower, designed as a three-wheeler with rear-wheel drive, is a battery-powered device that incorporates two motors connected to the back wheels.


To enable IoT control, a NodeMCU ESP8266 microcontroller is utilized, and two L298N Motor Drive units allow users to adjust the mower's speed, 
direction, and blade activation remotely using the Blynk app. In terms of automation, the mower is equipped with essential sensors including
ultrasonic, PIR, and DHT22, which provide data to the microcontroller for obstacle detection, avoidance, and boundary maintenance within the 
lawn perimeter. 


By combining these components and functionalities, the project aims to enhance the efficiency and convenience of 
lawn maintenance while incorporating modern IoT capabilities for seamless control.


The fabricated lawn mower provides a more affordable and convenient solution, reducing labour and mowing time while eliminating the 
need for installations. It overcomes the drawbacks of existing lawnmowers, offering an efficient and cost-effective alternative 
for effective lawn maintenance.
